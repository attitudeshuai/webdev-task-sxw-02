# 页面截图

本目录存放项目在前端运行后的页面截图，用于任务参考、设计核对与文档展示。

## 截图环境

- 浏览器：Chromium（Playwright 无头模式）
- 视口：1440 × 900
- 截图方式：完整页面（fullPage）
- 项目运行地址：http://localhost:3000
